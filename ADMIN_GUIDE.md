# Руководство администратора

## Доступ к админ-панели

Админ-панель доступна только пользователям с ролью `admin`. После входа в систему кнопка "Админ" появится в навигационном меню.

**Учётные данные администратора по умолчанию:**
- Логин: `Admin`
- Пароль: `admin148`

## Функции админ-панели

### 1. Управление пользователями

#### Создание нового пользователя
1. Перейдите на вкладку "Пользователи"
2. Заполните форму:
   - **Фамилия** (обязательно)
   - **Имя** (обязательно)
   - **Отчество** (необязательно)
   - **Роль**: Пользователь или Администратор
3. Нажмите "Создать пользователя"
4. Система автоматически сгенерирует:
   - Логин (8 символов: буквы и цифры)
   - Пароль (8 символов: буквы и цифры)
5. Сохраните учётные данные - они будут показаны только один раз!

#### Удаление пользователя
1. В списке пользователей найдите нужного
2. Нажмите кнопку "Удалить"
3. Примечание: главного администратора (Admin) удалить нельзя

### 2. Очередь на одобрение видео

На этой вкладке отображаются все видео, загруженные пользователями и прошедшие автоматическую валидацию.

#### Одобрение видео
1. Просмотрите информацию о видео:
   - Превью
   - Имя файла
   - Пользователь, загрузивший видео
   - Дата загрузки
   - Параметры (разрешение, длительность, размер)
2. Нажмите "✓ Одобрить"
3. Пользователь получит уведомление об одобрении

#### Отклонение видео
1. Нажмите "✗ Отклонить"
2. В появившемся окне укажите причину отклонения (необязательно)
3. Пользователь получит уведомление с указанной причиной

### 3. Рассылка уведомлений

Отправка уведомлений пользователям системы.

#### Отправка уведомления
1. Выберите получателя:
   - "Всем пользователям" - массовая рассылка
   - Конкретный пользователь - индивидуальное уведомление
2. Введите заголовок уведомления
3. Введите текст сообщения
4. Нажмите "Отправить уведомление"

## Типы уведомлений

Система автоматически создаёт уведомления следующих типов:

- **Одобрено (✓)**: Видео пользователя одобрено администратором
- **Отклонено (✗)**: Видео пользователя отклонено с указанием причины
- **Информация (ℹ)**: Общие информационные уведомления от администратора

## Хранение данных

Все данные хранятся в `localStorage` браузера:
- `users` - список пользователей
- `adminQueue` - очередь видео на одобрение
- `notifications` - уведомления пользователей
- `videoHistory` - история обработанных видео

## Безопасность

⚠️ **Важно:**
- Сгенерированные логины и пароли показываются только один раз при создании
- Сохраняйте учётные данные в безопасном месте
- Главного администратора (Admin) нельзя удалить из системы
- Все действия администратора видны пользователям через уведомления

## Навигация

Из админ-панели доступны все разделы системы:
- **Главная** - загрузка видео
- **История** - история обработанных видео
- **Уведомления** - личные уведомления администратора
- **Админ** - текущая панель
- **Выход** - выход из системы
