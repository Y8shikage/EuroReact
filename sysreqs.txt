Система проверки видео:
Пользователь загружает видео в форму на Главной странице. JS анализирует Длительность видео, Разрешение, Контейнер. Далее видео передаётся через туннель ngrok (ссылка) в папку с проектом на Python. Активный файл Python анализирует полученное видео, сверяя параметры из файла JSON (Кодек и FPS) и формирует отчёт в формате JSON. После проверки файл с результатами отправляется обратно на сайт и выводится результат в соответствующем окне для параметра видео.
